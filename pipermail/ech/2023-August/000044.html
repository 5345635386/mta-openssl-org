(I have a question to ask in my next mail but figured it
might be good to send a bit of an update for context...)

At last month's IETF meeting Dennis Jackson from Mozilla
had a presentation [1] about results from ECH testing so
far with FF, chrome and cloudflare. Bottom line is that
all seems well, so the TLS WG plan now is to move forward
with final spec tweaks then publication of the ECH spec
that might become an RFC early in the new year. (Plans,
eh?;-)

Anyway, that means I'll be planning to try produce a PR
with my ECH implementation in roughly the same timeframe.
Be happy to continue our earlier chats about APIs etc
before then, if the project have made any progress on that
"HOWTO accept big inputs" issue. (Was there?)
