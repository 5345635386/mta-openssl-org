I have successfully installed Godaddy Wildcard SSL certificate in 9 units
of Hikvision DS-K1TA70MI-T / DS-K1T341AMF Face Recognition Terminal Door
Access Systems on 27 Oct 2022 Thursday.

Login to the web GUI.

Go to System > Security > Certificate Management

Import Passwords - refer to SSL Private Key
============================================

Certificate Type: HTTPS
Upload Certificate: plain.txt

Click Install.

Import Communication Certificate - refer to chained SSL certificate from
commercial certification authority
============================================================================================================

Certificate Type: HTTPS
Upload Certificate: chain.crt

Click Install.

Import CA Certificate - self explanatory
=========================================

Custom ID: maindoor
Upload Certificate: gd_bundle-g2-g1.crt

Click Install.

Reboot Hikvision Face Recognition Terminal. You MUST reboot for changes to
take effect!

You should now be able to access https://hikvision.teo-en-ming-corp.com

I have done this for an investment company at Keppel Road, Singapore.
