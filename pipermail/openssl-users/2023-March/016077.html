And it does not actually work.

 - github supports editing issues but edits are not sent, only initial
   posting
 - AFAIK github does not support replying through mail
 - github might have an API but how many MUAS support importing github
   issues from gihub? The API for mailing list archives called 'download
   mailbox' is widely supported by MUAS which means you can read the
   archived questions and answers locally in your MUA if the archive
   website accessibility, search, or other features are not to your
   liking.
