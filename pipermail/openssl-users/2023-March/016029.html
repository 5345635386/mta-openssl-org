On Fri, Mar 17, 2023 at 11:04:57AM +0100, Tomas Mraz wrote:

> We are seeking input on this potential change if you are currently
> actively contributing on openssl-users mailing list discussion and you
> believe such a change would substantially reduce your participation in
> the project.

This is not a good idea, and would be a great loss to the user
community.  Unlike Github, questions here are seen by all kinds of
users, some who are just "lurkers", learning interesting tidbits from
various topics as they go by, others who are specialised in specific
topics on which they can give insightful answers, but don't have the
time or energy to track Github issues, ...

Unlike Github issues, the list also makes it easy to post longer
explanatory context, track conversation threads trees, ...

I, for one, subscribe and contribute to the list, but do not and will
not track new Github issues.

Bottom line:  DON'T DO THIS.
