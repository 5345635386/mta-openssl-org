Revision 2.0.9 of the OpenSSL FIPS Object Module v2.0 (FIPS 140-2
validation certificate #1747) recently received formal approval:

  http://csrc.nist.gov/groups/STM/cmvp/documents/140-1/140val-all.htm#1747

This latest revision adds five new formally tested platforms
("Operational Environments") bringing the total number of such platforms
to 102; more by far than for any other validated cryptographic module.

We have some more platforms on the way...

As always, there is no need to upgrade any already deployed instances of
the OpenSSL FIPS Module. But, if you're doing new development it's
preferable to use the latest revision as that will be valid for all
tested platforms.
