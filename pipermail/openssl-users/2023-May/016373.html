   OpenSSL version 1.1.1u released
   ===============================

   OpenSSL - The Open Source toolkit for SSL/TLS
   https://www.openssl.org/

   The OpenSSL project team is pleased to announce the release of
   version 1.1.1u of our open source toolkit for SSL/TLS.
   For details of the changes, see the release notes at:

        https://www.openssl.org/news/openssl-1.1.1-notes.html

   Specific notes on upgrading to OpenSSL 1.1.1 from previous versions are
   available in the OpenSSL Migration Guide, here:

        https://www.openssl.org/docs/man1.1.1/man7/migration_guide.html

   OpenSSL 1.1.1u is available for download via HTTPS and FTP from the
   following master locations (you can find the various FTP mirrors under
   https://www.openssl.org/source/mirror.html):

     * https://www.openssl.org/source/
     * ftp://ftp.openssl.org/source/
