Corretto-8 is an implementation of Java 8 by Amazon. It should have the same set of ciphers as Oracle's Java 8 and thus the same compatibility as their JVM. If you encounter a compatibility issue, open a case with Amazon. I have not seen any compatibility issues in my own experience with both.
