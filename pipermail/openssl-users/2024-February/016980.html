the primary point to post technical questions about OpenSSL is GitHub
Discussions on the OpenSSL project:

https://github.com/openssl/openssl/discussions/categories/q-a

As for the example code for RSA signatures there is some in the demos
subdirectory of the source tree. The example uses RSA-PSS but that can
be fairly easily adapted to RSA PKCS#1 1.5 signatures.

https://github.com/openssl/openssl/tree/master/demos/signature
