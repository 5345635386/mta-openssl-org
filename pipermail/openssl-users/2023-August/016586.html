That comes from this code in the header file:

/* Below 0.9.8 is unacceptably low */
# if OPENSSL_API_LEVEL < 908
#  error "OPENSSL_API_COMPAT expresses an impossible API compatibility

So, for some reason, it believes you are attempting to get compatibility 
with OpenSSL API before 0.9.8.

Do you define the OPENSSL_API_COMPAT macro anywhere - and if so with 
what value?

include/openssl/configuration.h?

This looks like an environmental issue somewhere
