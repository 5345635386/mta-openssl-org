Your step (4) to me appears to be wrong.  If you are to install openssl to the following directory:

                C:\Program Files (x86)\OpenSSL\

then you need to do:


?  perl Configure VC-WIN32 -prefix=C:\Program Files (x86)\OpenSSL

When I build openSSL I use nasm, so after the above I do:


?  ms\do_nasm

?  nmake -a -f ms\nt.mak

?  nmake -f ms\nt.mak test

?  nmake -f ms\nt.mak install

Does that help at all?
