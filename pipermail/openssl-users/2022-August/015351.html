Hi , guys




   I have use the follow openssl commands to create a key pair for android app signing.

   Follow this link https://source.android.com/docs/core/ota/sign_builds#manually-generating-keys




 # generate RSA key

openssl genrsa -3 -out temp.pem 2048
Generating RSA private key, 2048 bit long modulus
....+++
.....................+++
e is 3 (0x3)


# create a certificate with the public part of the key
openssl req -new -x509 -key temp.pem -out releasekey.x509.pem -days 10000 -subj '/C=US/ST=California/L=San Narciso/O=Yoyodyne, Inc./OU=Yoyodyne Mobility/CN=Yoyodyne/emailAddress=yoyodyne at example.com'


# create a PKCS#8-formatted version of the private key
openssl pkcs8 -in temp.pem -topk8 -outform DER -out releasekey.pk8 -nocrypt


# securely delete the temp.pem file
shred --remove temp.pem
