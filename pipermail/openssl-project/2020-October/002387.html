Given the fact that only one person asked for it and that only ten days are left in October,
I think the label solution might be the simpler one for the moment. (Assuming that adding
Topics to the project requires a vote?) The label has already been created and is ready
to be applied, to make the person happy which asked for it. In November, the label can
be removed again.

In general, I agree that GitHub Topics are a better solution. If we decide to participate
officially next year, we should take the time to inform ourselves in advance and announce
