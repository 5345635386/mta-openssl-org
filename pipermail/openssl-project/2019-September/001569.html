I wanted to ask if the plan (mentioned here: https://www.openssl.org/blog/blog/2018/09/25/fips/) to release OpenSSL version 3.0 and its FIPS 140-2 validated module, before EOY 2019 is progressing as expected, or should we expect a delay for that release?
