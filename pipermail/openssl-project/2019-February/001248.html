The vote text says: 
  
CCLAs and ICLAs will not be accepted unless they include all 
non-optional details.  The fields which are considered optional are 
public name and telephone for an ICLA and telephone and fax for a CCLA. 
  
The vote passed with five +1's and two abstentions. 
  
What does this mean for existing CLAs?  Nothing.  They are still valid. 
For CLAs going forwards, we'd prefer that all fields are provided but will accept forms with the some or all of the specified fields left blank.

We will not accept CLAs with any other field left blank.
