    Fix CRYPTO_strdup
    
    The function CRYPTO_strdup (aka OPENSSL_strdup) fails to check the return
    value from CRYPTO_malloc to see if it is NULL before attempting to use it.
    This patch adds a NULL check.
    
    RT3786
