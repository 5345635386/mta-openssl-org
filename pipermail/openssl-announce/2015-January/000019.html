   OpenSSL version 1.0.2 released
   ===============================

   OpenSSL - The Open Source toolkit for SSL/TLS
   http://www.openssl.org/

   The OpenSSL project team is pleased to announce the release of
   version 1.0.2 of our open source toolkit for SSL/TLS. For details
   of changes and known issues see the release notes at:

        http://www.openssl.org/news/openssl-1.0.2-notes.html

   OpenSSL 1.0.2 is available for download via HTTP and FTP from the
   following master locations (you can find the various FTP mirrors under
   http://www.openssl.org/source/mirror.html):

     * http://www.openssl.org/source/
     * ftp://ftp.openssl.org/source/
